📱 Einrichtung (einmalig)

Damit der Kamera-Scanner funktioniert, muss die App über HTTPS aufgerufen werden. Wähle eine Option:

Option A – Kostenlose Online-URL (empfohlen)

Skript deploy_github.bat doppelklicken → ergibt eine permanente HTTPS-URL wie
https://deinname.github.io/dexis-scanner
Funktioniert auf allen Geräten, kostenlos, für immer.

Option B – Nur im Heimnetzwerk

Skript local_server.py doppelklicken → QR-Code erscheint im Fenster → Mit Handy scannen (Browser-Warnung einmalig bestätigen).

Option C – Kurz testen (temporär)

Skript tunnel.py doppelklicken → Temporäre HTTPS-URL erscheint → Link öffnen (gilt nur solange Fenster offen).

ℹ️ Kamera funktioniert nur über HTTPS oder localhost – nicht über file://
